# 🌐 Areen Ibrahim – Portfolio

A modern personal portfolio website built with React, TypeScript, and Vite to showcase my projects, skills, and experience as a software engineering graduate.

## ✨ About the Project

This portfolio represents my work as a frontend-focused developer passionate about building clean, responsive, and user-centered digital experiences. It highlights my academic and practical projects, including web and mobile applications developed during my studies.

The goal of this portfolio is to present my skills, projects, and growth as a developer in a simple and visually engaging way.

## 🛠️ Built With

- React
- TypeScript
- Vite
- Tailwind CSS
- Motion (Framer Motion alternative)
- Lucide Icons

## 📂 Features

- Responsive design for all devices
- Smooth animations and modern UI
- Projects showcase section
- About me section
- Contact form UI
- Clean and minimal design

## 📌 Purpose

This project was created as part of my learning journey and to present my skills as a frontend developer specializing in UI/UX and modern web development.
